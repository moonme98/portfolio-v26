PROJECT DETAIL

WINGSTEP은 러닝 기록을 측정하고 관리할 수 있도록 설계한 Full-Stack 기반 웹 서비스입니다. React를 활용하여 사용자 인터페이스를 구현하고, Spring Boot REST API 서버에서 인증 및 데이터 처리를 담당합니다.

JWT 기반 인증 구조를 적용하여 로그인 상태를 관리하였으며, 사용자·코스·결제·운동기록 데이터를 MySQL에 저장하고 MyBatis를 통해 연동하였습니다. 전체 API 흐름을 직접 설계하고 구현한 프로젝트입니다.
